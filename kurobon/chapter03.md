### 問11
- equalsメソッド
同値性を確認するメソッド
null以外の参照値Xがあったとき、x.equals(x)はtrueを返す
null以外の参照値Xとyがあったとき、y.equals(x)がtrueを返す場合は、x.equals(y)はtrueを返す
null以外の参照値xとy,zがあったとき、x.equals(y)がtrueを返し、y.equals(z)がtrueを返す場合は、x.equals(z)はtrueを返す
null以外の参照値xとyがあったとき、x.equals(y)を複数回呼び出しても、比較で使われた情報が変更されていなければ、一貫してtrueを返すか、一貫してfalseを返す
null以外の参照値xについて、x.equals(null)はfalseを返すこと
nullがあればfalseを返す

### 問12
- ビット演算子
コンピュータの基本的な情報単位であるビット（０または１）を直接操作する演算のこと

| 演算子 | 演算 | 説明 |
|-------|-----|-----|
| & | AND | 両方のビットが1の場合に１を返す |
| | | OR | 片方または両方のビットが1の場合に１を返す |
| ^ | XOR | 片方のビットが1の場合に１を返す |
| ~ | NOT | ビットを反転させる |
| << | 左シフト | ビットを左にシフトする |
| >> | 右シフト | ビットを右にシフトする |
| >>> | ゼロ埋め右シフト | 符号に関係なく、左側をゼロで埋めながら右にシフトする |
